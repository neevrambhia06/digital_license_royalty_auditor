# Digital License Royalty Auditor

Full-stack AI-powered platform for detecting royalty leakage.

## Sprint 1 Features
- Supabase Schema Setup
- Synthetic Data Generation (100k+ logs)
- Backend/Frontend Project Scaffolding

## Tech Stack
- React + Vite
- FastAPI
- Supabase
- Agentic AI (Python)

## Getting Started
1. Configure `.env` files.
2. Run `python generate_data.py`.
3. Start backend: `cd backend && pip install -r requirements.txt && uvicorn main:app --reload`.
4. Start frontend: `cd frontend && npm install && npm run dev`.
